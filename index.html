🔥 Firebase Setup Required

To enable live scores across all devices, connect to a free Firebase Realtime Database.
Don't have one? Follow the steps below — it takes about 3 minutes.

1. Go to console.firebase.google.com → Create a project (disable Google Analytics is fine).
2. In the left sidebar → Build → Realtime Database → Create database → choose Start in test mode.
3. Click the ⚙ gear icon → Project settings → scroll to Your apps → click </> Web → register app.
4. Copy the config values into the fields below and click Connect.
